# 📊 SQL-Based Sales Analytics

A **MySQL-based Sales Analytics** project that transforms raw customer, product, and sales data into meaningful business insights through dimensional modeling and advanced SQL analytics.

The project demonstrates how SQL can be used to build analytical reports, evaluate business performance, and support data-driven decision-making.

---

## 📌 Project Overview

This project integrates customer, product, and sales datasets into a **dimensional (star) schema** and performs end-to-end sales analytics using MySQL.

The analysis answers key business questions such as:

- Which customers generate the highest revenue?
- Which products contribute the most to sales?
- How do sales change over time?
- Which products should be promoted or discontinued?
- How can customers be segmented based on purchasing behavior?
- What are the overall sales trends and KPIs?

## 📂 Database Schema

The project follows a **Star Schema** consisting of:

### Fact Table

- `fact_sales`

### Dimension Tables

- `dim_customers`
- `dim_products`
